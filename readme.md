## Description
A set of header only algorithms, data structures, and utilities I use in many of my other projects.  Most should have tests located in the [tests](tests/) folder

## Building
to build
``` bash
mkdir build
cd build
cmake ..
```

## Testing
To run unit tests

``` bash
make check
```
